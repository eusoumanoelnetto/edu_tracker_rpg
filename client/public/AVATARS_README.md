# Avatares do RPG Edu Tracker

## Avatares Padrão Incluídos
- `character-sprite.png` - Avatar padrão do aventureiro

## Como Adicionar Mais Avatares

Para adicionar novos avatares ao sistema, basta:

1. Adicionar as imagens no formato PNG (64x64px ou 128x128px) nesta pasta `/public`
2. Nomear seguindo o padrão: `avatar-{nome}.png`
3. Editar o arquivo `EditProfileDialog.tsx` e adicionar o caminho na constante `AVATAR_OPTIONS`

### Sugestões de Avatares:
- `avatar-warrior.png` - Guerreiro
- `avatar-mage.png` - Mago
- `avatar-rogue.png` - Ladino
- `avatar-knight.png` - Cavaleiro
- `avatar-scholar.png` - Estudioso
- `avatar-ninja.png` - Ninja
- `avatar-paladin.png` - Paladino
- `avatar-archer.png` - Arqueiro

### Recursos para Encontrar Sprites:
- [OpenGameArt.org](https://opengameart.org/)
- [Itch.io Game Assets](https://itch.io/game-assets/free)
- [Kenney.nl](https://kenney.nl/assets)

### Formato Recomendado:
- **Tamanho**: 64x64px ou 128x128px
- **Formato**: PNG com transparência
- **Estilo**: Pixel art ou cartoon
- **Paleta**: Combinar com o tema retro/arcade do app
